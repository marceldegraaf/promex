    decrement(name)

    Decrement the gauge identified by name by 1, or initialize it with a value of -1.

    iex> Promex.Gauge.decrement("test_gauge_4")
{:ok, -1}
iex> Promex.Gauge.decrement("test_gauge_4")
{:ok, -2}
    decrement(name, list)

    Decrement the gauge identified by name by a specific value.

    By default, this will set the gauge value to -1. To set a different initial value, use the by option.

    iex> Promex.Gauge.decrement("test_gauge_5", by: 5)
{:ok, -5}
iex> Promex.Gauge.decrement("test_gauge_5", by: 5)
{:ok, -10}
